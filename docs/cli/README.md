# Nestor CLI

The `nestor` CLI is the command surface for the Nestor memory API. It starts
with a compact command map, then points to command, workflow, slot, and error
details as needed.

Shortest path:

```sh
nestor guide
nestor doctor
nestor guide workflow
```
